# Order Connect

An application for order management for small industries that outsource machining orders to small machine shops.

## scope of the application

- The small industries can search by

  - available machines or
  - select a particular vendor based on rating etc or
  - publish the job to multiple vendors and request for quote which is then processed

- The machine shops can list available machines and specify the list of orders undertaken

- Tracking of order and its status is done.

- Delivery of goods to destination is tracked

## Technologies proposed for application

- Springboot
- Golang
- Reactjs
- Vuejs
- ReactNative
- Kafka
- RabbitMQ
